**Friendship Zone Web Application**
A simple web application built using HTML that allows users to connect with friends, view friend lists, and register themselves on the platform.

**Project Overview**
Friendship Zone is a beginner-friendly web project designed to simulate a social connection platform. It includes multiple static pages that demonstrate navigation, form handling, and structured UI design using HTML.

**Pages Included**
Homepage.html
→ Landing page of the application.
FriendList.html
→ Displays the list of existing friends in the Friendship Zone.
RegistrationForm.html
→ Allows new users to register by filling out a form.
RegistrationConfirmation.html
→ Confirmation page shown after successful user registration.
RegistrationConfirmationforAdmin.html
→ Confirmation page for admin registration.

**Features**
Simple and clean UI using pure HTML
Multiple interconnected pages
User registration form
Confirmation pages for both users and admin
Easy navigation between pages

**Technologies Used**
HTML5

**Learning Outcomes**
Understanding of basic web page structure
Creating and linking multiple HTML pages
Designing forms using HTML
Organizing a simple web project
